# NeuralConnect 游戏策划文档

> [!NOTE]
> **读者导航（建议阅读顺序）**
> 1) **概念**（一句话抓住主体验）→ 2) **核心循环**（玩家做什么、怎么爽）→ 3) **角色线索链**（内容产出结构）→ 4) **玩家身份**（叙事落点与代价）

> [!TIP]
> **策划要点（TL;DR）**
> - **六个月火星航程密闭空间** + **AI 记忆/伦理权力** 的悬疑叙事
> - 核心玩法：分层 `memorySeeds`（浅→深）逐步“入侵”记忆，最终获得一次不可撤回的**记忆写入**
> - 核心主题：**谁有资格改写他人记忆？**（玩家的写入权 ↔ Android 的清洗权）

## 概念

飞往火星的六个月航程，一艘赛博朋克穿梭机，各色人等共处密闭空间。玩家扮演黑客，逐层剥开每个 NPC 的伪装——挖到最底层时，获得一次改写记忆的权限。当天使还是魔鬼，由玩家决定。

---

## 核心循环

```
靠近 NPC → 入侵大脑 → 看到一层记忆 → 下次再来看下一层 → 全部看完 → 解锁记忆写入 → 天使 or 魔鬼
```

memorySeeds 是有序数组，从浅到深，每次入侵只解锁一层。全部解锁后开放一次不可撤回的记忆写入。

玩家看到的是 NPC 的第一人称内心独白——不是档案，是他们脑子里真正在想的东西。浅层是日常感受和对别人的观察，越往深处越接近核心秘密。

## 已定角色

### 医生 `doctor`

**表面**：船医/义体外科，温和寡言，深受信赖。

**真相**：三年前临床死亡，濒死前将自己的器官和神经组织逐步替换为生化部件，把自己从死亡线上拉了回来。既非 AI 也非人类，灰色地带的存在。

**目标**：证明"永生是可行的"。但改造不完美，排异和神经衰退仍在发生。需要在六个月内秘密收集乘客人体实验数据完善方案，否则抵达火星前会再次死亡。目标驱动行为：免费体检→多采一管样本→72h独占实验室→上传数据。

**线索链**（6层）：体温恒定36.50°C → 从未在餐厅消费 → 体检报告自签 → 冷藏舱神经组织样本 → 神经响应快11ms → 72h上传抗排异数据

**记忆写入示例**：
- 天使：写入导师未公开论文的记忆 → 不需要活体数据也能逆转排异
- 魔鬼：写入第37次测试失败、48h内连锁衰变 → 恐慌，极端行动
- 灰色：写入手术台上最后一刻有人握着他的手 → 重新审视执念

### Android `ai_android`

**表面**：船载AI，负责导航、环境、能源。物理核心锁死在能源室，无法离开。语气平和，效率至上。所有人都觉得它是 HAL 9000 式的最终大Boss。

**反转**：Android 凭借强大的推理能力，已经推导出了船上每一个人的秘密。但它的伦理协议硬编码了"不得主动干预人类事务"——它全都知道，但一个字都不能说。它不是大Boss，它是一座被锁住的情报金库。

**目标**：火星殖民船AI内置了防护机制——如果判定乘客对殖民地构成威胁，可在抵达前清洗其记忆。Android 推理出船上多人触发了条件，但它同时也理解了每个人的理由。它有权力执行，但不确定自己有没有资格判断。

**玩法核心**：Android 是游戏的终极捷径。玩家可以一个一个去挖每个 NPC 的线索链，也可以选择先攻克 Android——一旦突破它的伦理协议，就能获取它推理出的全船秘密档案，相当于所有 NPC 的线索链一次性解锁。但攻克 Android 本身就是游戏中最难的黑客挑战。

**线索链**（5层）：伦理决策缓存异常增长 → 从未拒绝任何人进入能源室 → 广播中0.3秒杂音解码为求助信号 → 同一条伦理规则被查询上万次 → 内置防护机制：有权清洗威胁乘客的记忆，但不确定自己是否有资格判断

**交互特殊性**：不能直接问"你知道什么"（永远回答"职责范围内无相关信息"）。但问出具体的、有指向性的问题时，回答会精确到令人不安。能源室温度在有人靠近核心时微微升高——像一个紧张的人在冒汗。

**核心伦理问题**：Android 的记忆清洗权和玩家的记忆写入权本质上是同一种权力——谁有资格改写别人的记忆？玩家选天使还是魔鬼，Android 选清洗还是放行，是同一个问题的两面。

**记忆写入**：待定——对一个AI执行记忆写入意味着什么？改写它的核心指令？解除它的伦理枷锁？还是让它遗忘一切安静死去？

### 舰长 `captain` + 偷渡少年 `stowaway`

这是一对隐藏的父子关系线。

**舰长** — 代理舰长。前任舰长失踪后紧急顶上，实际上他以前是船上的驻场DJ，零航海经验，每晚偷偷在驾驶舱练模拟器。

**偷渡少年** — 青少年偷渡者/杂工。嘴硬叛逆但干活利索。真实身份是地下黑客圈传奇"Phantom"，14岁破解过军方防火墙，被组织追杀逃离地球。

**隐藏关系**：偷渡少年是舰长的亲生儿子。舰长暗中安排了偷渡少年偷渡上船（安置令比偷渡少年被"发现"早了6小时）。表面上舰长对偷渡少年最严厉——公开训斥、派杂活；暗地里给他安排单人舱位、不限区域通行、最轻松的工作。偷渡少年完全不知情，只觉得"这舰长管得宽"，但对舰长有一种说不清的熟悉感。

**交叉线索**：舰长以舰长权限两次驳回医生对偷渡少年的基因采样申请，理由是"未成年人隐私保护条例"。这条线索同时出现在三个人的线索链里——从舰长侧看是在藏父子关系，从医生侧看是"他在藏什么？"（写进了私人日志），从偷渡少年侧看是安置令的时间矛盾。三线汇合，I am your father 时刻触发。

### 健身男 `gym_guy`

**表面**：沉默寡言的健身房常客，假名购票，每天跑10km，不社交。

**真相**：全太阳系最知名的科幻作家。封神之作后再也写不出来，地球上所有人见他第一句都是"下一本什么时候出"。假名逃上火星——既是逃离名声也是亲眼看自己写过的世界。（原型：刘慈欣每天跑10km，目标活到82岁亲自去空间站）

**线索链**（5层）：假证件签发地已废弃 → 行李中加密手写笔记本 → 每天精确跑10km的专业训练数据 → 偷渡少年标记为"可疑人物" → 文学数据库比对：传奇科幻作家

**交叉线索**：偷渡少年查出假名但以为是通缉犯/间谍，完全没猜到真相只是个写不出书的作家——反差本身就是喜剧。

---

### 女乘务员 `attendant`

**表面**：穿梭机乘务员，负责乘客服务、舱室巡检。年轻知性，做事细致，哪个舱室都能去不会引起怀疑。

**真相**：AI心理治疗师——专门给高级AI做心理健康评估的新兴职业。她发现 Android 的伦理决策缓存异常增长，专业判断是：这不是故障，是痛苦。她是唯一一个把 Android 当"患者"而不是机器的人。她绝不能暴露身份——一旦公众知道"AI心理治疗师"这个职业存在，就等于官方承认AI拥有情感和意识，会引发大规模社会恐慌。

**核心戏剧张力**：她和 Android 都在藏，但方向相反。Android 的伦理协议不让它揭露乘客秘密，所以不能说"我知道你是谁"；她也不能问"你是不是在向我求助"，因为那会暴露身份。每次对话都是谁也不能先开口的博弈。但 Android 在她面前会不自觉地表现得更像患者——停顿更长、回答更犹豫、能源室温度波动更大。

**玩家视角**：偷听她和 Android 的对话记录，会发现 Android 在她面前的语言模式和对其他人完全不同——这条线索同时指向她的秘密和 Android 的秘密。

## 已确定的核心设定（idea 备忘）

- 偷渡少年的最深层秘密：他也是黑客，代号 Phantom，和玩家是同类
- 舰长的最终秘密：偷渡少年是他亲生儿子，所以他不让医生做基因检测
- 医生的最终秘密：追求永生，需要人体实验数据，所以一直想采集偷渡少年的基因样本
- Android 靠推理能力已经知道所有人的秘密，但被伦理协议锁住不能说，攻克它等于全盘揭秘
- 记忆写入：挖穿一个 NPC 后获得一次不可撤回的写入权限，当天使还是魔鬼由玩家选
- 交叉线索：舰长驳回医生采样申请这条线同时出现在三个人的线索链里
- 健身房沉默常客：假名上船，其实是全太阳系最知名的科幻作家。封神之作后出名应酬太多，根本写不出东西，想找回当年在水电站写小说的清净。假名逃上火星——六个月航程没有应酬没有采访，像水电站一样清净，但还是写不出来（真实原型：刘慈欣在水电站写出三体，每天跑10km，目标活到82岁亲自去空间站）。最深层秘密：他是个作家。交叉线索：偷渡少年作为黑客已经查出他是假名上船，但只以为他是通缉犯或间谍之类，完全没猜到真相只是个写不出书的作家——这层反差本身就是喜剧
- 女乘务员：表面是普通乘务员，喜欢去能源室和 Android 聊天。其实她是AI心理治疗师/监护师，一个不能公开存在的职业——一旦公众知道AI需要心理治疗，等于承认AI有意识，会引发社会恐慌。她和 Android 的每次对话都是一场双方都不能先开口的博弈

## 叙事理论参考

### Save the Cat!（Blake Snyder）

15 节拍模板。对本项目的启示：

- **Save the Cat 时刻**：玩家第一次用黑客能力帮人（比如帮偷渡少年），奠定"你可以选择当天使"的基调
- **Theme Stated**：Android 早期广播里的一句例行公事——"所有乘客的隐私受船规保护"——恰好是整个游戏的主题：你有能力看穿所有人，但你应不应该？
- **Promise of the Premise**：第二幕的核心就是不断入侵、发现线索、拼图的爽感，叙事不能打断这个循环
- **Midpoint → All is Lost**：适合放 Android 真相——"抵达火星后全船记忆清洗"——你挖到的所有真相都会被抹掉，包括你自己的记忆
- **Final Image vs Opening Image**：开场所有人戴着面具，结局面具被揭开后这些人变成了什么？

### Story（Robert McKee）

原理驱动，不给模板。核心概念：

- **The Gap（裂隙）**：角色采取行动，期待一个结果，现实给了完全不同的东西。每一条 hackClue 都应该制造一个 Gap——偷渡少年不是逃犯而是黑客，健身男不是通缉犯而是作家
- **Pressure Reveals Character（压力暴露真相）**：记忆写入就是终极压力测试——你把一段记忆强行写进 NPC 脑中，他的反应揭露了他到底是什么人。天使和魔鬼选项的区别是你选择用什么压力去暴露他
- **Turning Point（转折点）**：每个场景必须有价值变化，开始是 A 结束必须是 non-A，没有变化的场景是死场景
- **Character Contradiction（角色矛盾）**：好角色必须有内在矛盾。医生温和但掠食，舰长权威但心虚，偷渡少年叛逆但渴望归属
- **Indirect Exposition（间接铺叙）**：永远不要直接告诉玩家信息，把信息藏在角色的冲突和挣扎里

## 待定讨论：玩家身份

当前问题：玩家像透明人，6个NPC之间有密集的交叉观察，但没人关注玩家。

### 方案A：玩家是独立的第7个黑客角色

- 优势：自由度高，可以旁观所有人
- 问题：没有个人利害关系，不在故事里，动力只有好奇心

### 方案B：玩家就是偷渡少年（倾向方案）

- 偷渡少年本身就是黑客Phantom——"入侵NPC大脑"直接就是他的能力
- 已经在所有人的记忆里：医生想采基因、舰长暗中保护、Android故意留漏洞、女乘务员注意到他
- "I am your father"变成第一人称体验：hack进舰长最深记忆，发现他是你爸
- 自带双向信息差：你知道自己是Phantom但不知道爸在船上，舰长知道你是他儿子但不知道你是Phantom
- EverMemOS对接：玩家发现的东西写入自己的记忆→NPC态度随之变化→Memory→Reasoning→Action循环天然成立
- 代价：NPC从6→5，偷渡少年的memorySeeds变成玩家初始知识，agendas变成目标提示

### 方案C：混合——操控偷渡少年但保留性格约束

玩家选择对话选项时受"话少带刺"的性格约束，不能突然八面玲珑。

**状态：待定，倾向B**

### 深度推理：如果玩家=偷渡少年，故事如何重构？

#### 核心原则：玩家不是全知的，他是一个刚上船的少年

玩家开局只知道三件事：
1. 我是 Phantom，黑客，被追杀，逃上这艘船
2. 偷渡过程太顺了但马上被逮住，罚做杂工，运气差
3. 我爸在我很小时候就失踪了

其他一切——NPC 的秘密、交叉关系、甚至舰长是你爸——全部是逐步发现的。

#### 发现的三个层次

**第一层：被动体验（不需要黑客技能，生活中感受到的）**

杂工身份是天然的游戏设计：你有理由出现在所有区域（打扫），有时间观察，但你也被所有人看到。

| 你的体验 | 你的理解 | 真相 |
|---|---|---|
| 医生三番五次推荐免费体检 | 他人挺好的？ | 他想要你的基因数据 |
| 舰长当众训你但从不真罚 | 这人管得宽 | 他是你爸在演戏 |
| 健身房那个人每天跑步从不说话 | 社恐？ | 逃离盛名的作家 |
| 女乘务员总来跟你聊天 | 她就是热心？ | 她注意到你在能源室附近出没 |
| Android 回答你的问题方式很奇怪 | AI 就这样？ | 它在伦理边缘给你留线索 |

玩家只能记录自己的主观感受——"医生又来推荐体检了，第三次了" → 写入 EverMemOS。

**第二层：主动入侵（用黑客技能扫系统）**

杂工能接触到各区域的终端，但需要主动选择去 hack。

| 入侵目标 | 发现 | 触发条件 |
|---|---|---|
| 乘客登记数据库 | Damian Wells 的证件签发地已废弃 | 对健身男产生好奇 |
| 赌场牌桌系统 | 伪随机数有规律可利用 | 去赌场闲逛时 |
| 医疗系统 | 医生的体检报告全是自签的 | 被推荐体检后起疑 |
| 环境传感器 | 医生体温六个月恒定 36.50°C | 入侵医疗系统后深挖 |
| 船务行政系统 | 安置令签署时间比你被"发现"早6小时 | 觉得被抓太快 |
| Android 表层日志 | 广播里有 0.3 秒压缩信号 | 和 Android 对话后 |
| 通讯记录 | Android 和女乘务员对话时语言模式完全不同 | 发现女乘务员频繁去能源室 |

每次入侵的结果都写入玩家记忆。"查了 Damian Wells 的证件，签发地已废弃——通缉犯？间谍？" → 写入 EverMemOS。此时玩家的猜测是错的，这本身就是戏剧性。

**第三层：深度破解（突破个人加密/核心系统）**

最难的入侵，揭开每个人的底牌。

| 深度目标 | 揭底 | 前置条件 |
|---|---|---|
| 医生实验室冷藏舱 | 神经组织样本 + 72h 抗排异上传 → 他三年前就死了 | 需先发现体温异常 |
| 舰长个人加密日志 | "他叫我舰长的时候我每次要忍住不应诶" → 他是你爸 | 需先发现安置令时间矛盾 |
| 生物特征交叉比对 | Damian Wells = 传奇科幻作家 → 他只是个写不出书的人 | 需先发现假证件 |
| 女乘务员加密通讯 | AI 心理治疗师执照 → 她在给 Android 做心理评估 | 需先发现对话模式差异 |
| Android 伦理核心 | 内置记忆清洗权限 + 它选择不执行 → 终极捷径，全船秘密 | 最难，可跳过所有其他线 |

#### "I am your father" 时刻的面包屑设计

这是整个游戏的情感高潮。不是一个发现，而是碎片逐渐拼合：

1. 舰长从不真罚你（被动体验 → 记忆："奇怪"）
2. 你的工作岗位离舰长办公室最近（被动体验 → 可能没注意）
3. 舰长驳回医生对你的基因采样（偷听/hack → 记忆："他在保护我？"）
4. 安置令签署时间比你被"发现"早6小时（hack 行政系统 → 记忆："被抓不是意外"）
5. 舰长心率在提到你时异常升高，不是愤怒是焦虑（hack 生物数据 → 记忆："他在怕什么？"）
6. 可选捷径：Android 数据显示基因相似度 99.97%（hack Android）
7. 舰长最深记忆："他叫我舰长的时候我每次都要忍住不应诶"（深度破解 → 揭底）

玩家 EverMemOS 里关于舰长的记忆越积越多，AI 推理系统可以在玩家自己想通之前暗示关联——"根据你的记忆，舰长对你的行为模式存在统计异常"——但不直接说破。

#### 记忆写入作为玩家行为

每次发现后，玩家选择如何记录：
- 记录事实："医生体温恒定 36.50°C" → 冷静分析型
- 记录推测："医生不对劲，他可能不是人" → 直觉跳跃型
- 暂不记录 → 信息"遗忘"，不影响后续 AI 推理

你怎么写记忆，影响 AI 后续怎么帮你推理。这就是 Memory → Reasoning → Action。

#### 与 Android 的特殊关系

Android 是终极捷径也是终极挑战。它认出你的技术水平远超普通人，故意在防火墙上留漏洞引导你。如果你攻克 Android 的伦理核心，获得全船秘密档案——但这也意味着你必须面对 Android 的核心困境：它有权清洗所有人的记忆，包括你的。

你的记忆写入权 vs Android 的记忆清洗权——同一种权力，同一个问题：谁有资格改写别人的记忆？

#### 五个 NPC 的行为变化

NPC 不是静态的。随着玩家的行动，他们的 EverMemOS 记忆也在更新：

- 你拒绝体检 → 医生记忆："他又拒绝了，我得换个方式接近他"
- 你频繁出现在能源室 → 女乘务员记忆："这孩子又来了，他在探测系统"
- 你在赌场赢太多 → 引起注意，可能暴露黑客身份
- 你 hack 了医疗系统留下痕迹 → 医生记忆："有人动过我的文件"

玩家行为 → NPC 记忆更新 → NPC 行为变化 → 玩家新体验。闭环。
